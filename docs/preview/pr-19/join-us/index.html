    横井研究室 @ 国語研での活動を希望するかたへ

    横井研究室 @ 国立国語研究所 (国語研) に何らかの方法でジョインしてみたいというかた向けの情報を集めたページです．今後も質問をいただく都度更新していきます．

    参加の仕方の選択肢

    横井研究室 @ 国語研にジョインいただく方法は色々あります．
    以下，主としてすでに先人がおられる方法を挙げますが，他にも様々な選択肢がありそうです．研究・仕事上のご興味や，関わり方に関する希望や制約について教えていただければ，何らかの新しい方法を提示できるかもしれません．まずは横井まで気軽にコンタクトをください．現在のラボメンバー一覧 (members) も参考になるかと思います．

    🎓 博士後期課程学生として進学したい

    総合研究大学院大学 (総研大) に入学してもらうことで，国語研で博士生活を送ることができます．
    総研大の「コース」に対して各国立研究所が紐づいており，総研大の日本語言語科学コースが，国語研に所属したい場合の進学先です．
    「日本語言語科学コース」という名前ですが，個別言語に全く限らず，広い意味で自然言語の数理について研究したいかたを受け入れます．

    毎年，国語研公式の相談会・説明会が開催されています．ご興味があるかたはぜひ気軽にご参加ください．

    • 毎年7月：国語研のオープンハウス「ニホンゴ探検」に，日本語言語科学コース 相談会 (大学院進学相談会) が併設されます．
    • 毎年9月：日本語言語科学コース 説明会が (大学院進学希望者向けの説明会) が開催されます．

    また，願書は毎年12月初旬（必着）です．検討されているかたはお忘れなきようご注意ください．

    🏫 他大学の学部・修士・博士学生として定期的に通い共同研究したい

    TBW
    「研究室に在籍している＊＊さんと一緒に研究をしてみたい」といった希望があればごく気軽に相談してください．数理的な手続き，およびアイデアを検討する議論の両方が好きなかたは特に楽しめると思います．

    👩‍🎓👨‍🎓 ポスドクとして就職したい

    学振PDでの就職を受け入れることができます．
    また，ラボ・研究所・機構それぞれで採用中のポスドクポジションはそのときどきで変わります．まずは横井まで気軽にご相談ください．

    🏢 他の機関で働きながら定期的に遊びに来たい

    外来研究員など，ラボでの研究や議論の環境を提供するための仕組みがあります．まずは横井まで気軽にご相談ください．

    👩🏻‍💻👨‍💻 バックオフィス・サポートメンバーとして研究室運営に参加したい

    広く研究室運営のサポートにご興味があるかたはぜひ横井までご連絡ください．
    2025年11月現在，4名のかたにバックオフィスメンバーとして入っていただいています．引き続き，環境作り，広報，情報システム，デザイン，などなど多くのかたをお雇いできればと考えています．
