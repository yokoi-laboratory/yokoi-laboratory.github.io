    横井研究室@国語研での活動を希望するかたへ

    横井研究室 @ 国立国語研究所 (国語研) に何らかの方法でジョインしてみたいというかた向けの情報を集めたページです．今後も質問をいただく都度更新していきます．
    ジョインいただく方法は色々あります．以下にすでに先例がある方法を挙げますが，他にも様々な選択肢がありそうです．研究・仕事上のご興味や，関わり方に関する希望や制約について教えていただければ，何らかの方法を提示できるかもしれません．まずは横井まで気軽にコンタクトをください．

    博士後期課程学生として進学

    総合研究大学院大学 (総研大) に入学してもらうことで，国語研（を含む多くの国立研究所）の研究室で大学院生活を送ることができます．
    総研大の「コース」とそれぞれの国立研究所が紐づいており，総研大の日本語言語科学コースが，国語研で博士後期課程を過ごしたい場合の進学先です． もともと言語学の研究所ですが横井研究室に進学される場合は数理的側面に「日本語」とつきますがこれに限る必要はなく，

    毎年，国語研公式の相談会・説明会が開催されます．ご興味があるかたは気軽にご参加ください．

    • 毎年7月：国語研のオープンハウス「ニホンゴ探検」に，日本語言語科学コース 相談会 (大学院進学相談会) が併設されます．
    • 毎年9月：日本語言語科学コース 説明会が (大学院進学希望者向けの説明会) が開催されます．

    願書は毎年12月初旬（必着）です．検討されているかたはお忘れなきよう．
